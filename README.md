# User Mobile Application

React Native mobile application for customers using Expo.

## Features

- User authentication
- Browse barbers
- Book appointments
- View appointment history
- Profile management
- Responsive design

## Setup

1. Install dependencies:
```bash
npm install
```

2. Update API URL in `src/lib/api.ts`

3. Run the app:
```bash
npm start
```

Then press `i` for iOS or `a` for Android.

## Tech Stack

- React Native
- Expo
- TypeScript
- React Navigation
- Zustand (State Management)
- Axios
